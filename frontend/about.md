# A2A Multi-Agent Frontend

**Type:** PRODUCTION
**Description:** Professional React frontend for the A2A Multi-Agent System. Replaces Gradio with a modern chat interface, real-time agent status, artifact viewer, and robust error handling.
**Stack:** React 19, Vite, TypeScript, Tailwind CSS v4, Framer Motion
**Backend:** Python FastAPI agents (ports 8000-8003) with A2A protocol
**Deployment:** Served by nginx in Docker container alongside agents

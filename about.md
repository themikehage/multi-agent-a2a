# A2A Multi-Agent System

**Type:** PRODUCTION
**Description:** Multi-agent system using the Agent2Agent (A2A) protocol with specialized agents for data analysis, planning, and creative content generation. Orchestrated by a host agent with a React + TypeScript frontend.
**Stack:** Python 3.12, FastAPI, React, TypeScript, Tailwind CSS, A2A Protocol
**Deployment Target:** Coolify (Docker single-container with supervisord)
**LLM Backend:** Qwen (DashScope API) via OpenAI-compatible endpoint
